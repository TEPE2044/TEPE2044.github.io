开始使用VitePress ​

0. 安装VitePress ​

确保你的电脑上已经安装了Node.js，开始构建VitePress项目

powershell
npm add -D vitepress

1. 生成VitePress ​

温馨提示：你最好在空文件夹中进行安装，因为VitePress的生成不会新建一个文件夹。

powershell
npx vitepress init

要不要用TypeScript🤨？总之我不选

3. 运行VitePress项目 ​

恭喜你，VitePress基础项目已搭建完毕，运行它

powershell
npm run docs:dev
